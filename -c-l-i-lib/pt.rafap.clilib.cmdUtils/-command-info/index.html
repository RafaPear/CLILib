CommandInfo

data class CommandInfo(val description: String, val usage: String, val aliases: List<String>, val longDescription: String = description, val minArgs: Int = 0, val maxArgs: Int = 0, val commands: List<String> = emptyList(), val requiresFile: Boolean = false, val fileExtension: String = "")

Immutable metadata describing a CLI command.

Parameters

description

Short, one-line description shown in help listings.

usage

Usage string demonstrating how to call the command.

aliases

List of aliases that can be used to invoke the command.

longDescription

Longer, detailed description shown in extended help. Defaults to description.

minArgs

Minimum number of arguments the command accepts.

maxArgs

Maximum number of arguments the command accepts; use -1 for unlimited.

commands

Optional list of sub-commands or fixed positional tokens supported by the command.

requiresFile

Whether the command expects a file path argument.

fileExtension

Expected file extension when requiresFile is true (empty means any).
